## 專案特色

- 純 `HTML / CSS / JavaScript`，無框架、無 build
- 可直接部署到 GitHub Pages
- 多款可直接遊玩的 Canvas 小遊戲

## 本機啟動

```bash
python -m http.server 8000
```

開啟：<http://127.0.0.1:8000/>
