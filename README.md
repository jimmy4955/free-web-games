# Free Web Games Hub

純 HTML/CSS/JS 的免費網頁小遊戲站，支援 GitHub Pages 直接部署。

## 已上線遊戲

- `渡劫修仙`
  路徑：`/game/tribulation-ascension/`

## 專案結構

```text
/
  index.html
  styles.css
  script.js
  /assets
    game-icon_tribulation-ascension.png
    icon_main-menu.png
  /game
    /tribulation-ascension
      index.html
      game.css
      game.js
      /assets
        bg.png
        /sprites
          idle-spritesheet-left.png
          idle-spritesheet-right.png
          walk-spritesheet-left.png
          walk-spritesheet-right.png
```

## 本機預覽

直接開啟根目錄 `index.html`，或使用 Live Server / `python -m http.server`。
